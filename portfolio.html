RelayFX Desktop Application

This application back-end is written in Java and its front-end is written in FXML using the JavaFX technology. It is a control solution designed to control target device remotely via a bluetooth or zigbee serial transreceiver module connected to the operational desktop computer via USB. The target device shall have an embedded board as an intermediary between its transreceiver and the device control switch board. If your embedded board is an arduino, You can as well download the arduino code for this project here. You are free to download this code, study and edit to suit you. The operational PC must run JRE 1.8 or above.

Fingerprint Attendance App

This application is written in Java using the swing technology. It is developed to basically solve proxy attendance taking in any institution. The application is designed to use only digital personna USB fingerprint hardware. The application uses JDBC to interract with localhost MySQL database to store enrolled candidates data. You as an administrator can enroll two fingers for each candidate, such candidate can use either of the finger to take his/her attendance for the day. It tabulate the attendance for that day upon query. You are free to download this code, study and edit to suit you. The user name and password is gateway.

Arduino code for automatic water pump system

This code is written in C++. It is an embedded program designed to automatically monitor liquid level in a tank and then smartly decides if to off the running pump or start the pump. The program is compactible with AVR microcontrollers. The microcontroller running this program controls the hardware that controls the pump. Real time monitoring result is displayed on Liquid crystal Display (LCD). You are free to download this code, study and edit to suit you.

Arduino code for college bell system

This code is written in C++. It is an embedded program designed to automatically monitor the calender of a typical school using the Real Time Clock (RTC) module DS1307, it automatically sounds an alarm to signify the end of every class period. It is smart enough to recognize saturday and sunday as weekend, hence does not sound alarm for weekends and friday alarm is quite different as per the schooling system in south-west region of Nigeria. Real time monitoring result is displayed on Liquid crystal Display (LCD). The program is compactible with AVR microcontrollers. The microcontroller running this program controls the hardware that controls the alarm sound system. You are free to download this code, study and edit to suit you.
