MMS RESTful web service

Development of RESTful web service for mini-grid management. Having served as utility manager at Rubitec Gbamugbamu Microservices, I realised some challenges and took a confident step of writting server side solution for the problem as RESTful web service. The project named "MMS" is developed to solve challenges of persisting data in to cloud. Resources such as shutdown, expenditure, generator runtime, site, operator, vendor e.t.c were created and endpoints were provided for client side to consume. Client can then download data in CSV format. This software makes it easy for the utility manager to persist record using mobile app or web app and be rest assured the headquater have access to it instantly or anytime needed. The project has service layer that performs some logics with the data before persisting. The program also send email notifications as necessary to all registered recipients of the operator. The project is developed using Java with Spring framework, lombok, Hibernate, Jackson, Kubernate etc.

Solar Mini Grid

Management of 85KWp solar mini-grid owned by Rubitec Nigeria Limited and located at Gbamugbamu village, Ijebu-Ode. I serve as the Utility Manager running the grid 24/7, ensuring utilization of generarted energy to create more revenue at minimal running cost. I coordinate the sales of power as well. My responsibility include solving problem as fast as possible to minimize down time. In the course of serving the company, I begin to develop custom mini-grid management software aimed at solving record persistance into remote database and making this record available for better decisions making. The software also automatically sends email notifications. I also introduced a software solution that allows mass shutdown and restoration of customers' meters, a functionality not readily available on the platform used by the company.
