PDF generation service

Pdf generation restful microservice that generates customer reports as PDF. The service has the capability of generating various types of reports for customers as specified by client requests. It aggregates data from over 23 API endpoints called and processed in parallel. Stacks used are Java 11, JAX-RS, Jetty server, Itext7, Hibernate, Oracle DB, H2 test DB, Swagger, JUnit, Mockito, Thymeleaf template engine, HTML5, Maven, Google Guice (DI), Jenkins, Docker, Red Hat Open-Shift, Bitbucket, Sonarqube, Jfrog.

Status: completed
Category: production
