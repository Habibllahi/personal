VTU SLOT

VTU slot is a project I entirely work on with Spectrangle technologies. This project is built using Spring/Java stack. Its uses Monolythic architecture and connects with third party services such as VerifyMe (for KYC validation)and Providus bank (For Wallet funding and management), Twillo (for management of app communication with client via SMS and Call). The project provideds a Peer2Peer solution for trading network airtime and Data. This project is hosted on Heroku Platform As Service.

Project MMS (RESTfu web serice)

Development of RESTful web service for minigrid management. The project named "MMS" is developed to solve persistent challenges of monitoring and management of data in to database. MMS is developed using Java, Spring framework, Hibernate, Jackson, lombok, MySQL RDBMS, e.t.c

Solar Mini Grid

Management of 85KWp solar mini-grid owned by Rubitec Nigeria Limited and located at Gbamugbamu village, Ijebu-Ode. I serve as the Utility Manager running the grid 24/7, ensuring utilization of generarted energy to create more revenue at minimal running cost. I coordinate the sales of power as well. My responsibility include solving problem as fast as possible to minimize down time. In the course of serving the company, I begin to develop custom mini-grid management software aimed at solving record persistance into remote database and making this record available for better decisions making. The software also automatically sends email notifications. I also introduced a software solution that allows mass shutdown and restoration of customers' meters, a functionality not readily available on the platform used by the company.

4 units Solar Installation

3.5KVA solar installation done at Abeokuta. It consist of 4 units 500watt sun-powered monocrystalline solar panel, 1 MPPT 80A charge controller, 4 units 12v/200Amphr Gel batteries, 3.5/48v Mikrotek inverter.

2 units Solar Installation

1KVA solar installation done at Sanyo, Ibadan. It consist of 2 units 150watt sun-powered polycrystalline solar panel, 1 Techfine 1KVA inverter, 40Amp MPPT charge controller, 2 units 12V 100Amphr Gel batteries. The system can still comfortably support one additional panel.
