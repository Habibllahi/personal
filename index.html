Featured Portfolio

RelayFX Desktop Application

This application back-end is written in Java and its front-end is written in FXML using the JavaFX technology. It is a control solution designed to control target device remotely via a bluetooth or zigbee serial transreceiver module connected to the operational desktop computer via USB.

Solar Installation

1KVA solar installation done at Sanyo, Ibadan. It consist of 2 units 150watt sun-powered polycrystalline solar panel, 1 Techfine 1KVA inverter, 40Amp MPPT charge controller, 2 units 12V 100Amphr Gel batteries. The system can still comfortably support one additional panel.

Arduino code for automatic water pump system

This code is written in C++. It is an embedded program designed to automatically monitor liquid level in a tank and then smartly decides if to off the running pump or start the pump. The program is compactible with AVR microcontrollers.
